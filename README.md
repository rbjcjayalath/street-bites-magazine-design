# 🍔 Street Bites - Magazine Design Project

A vibrant and modern multi-page street food magazine design concept created as part of the Graphics, Multimedia, and Social Media course module at IMBS Green Campus.

---

## 📌 Project Overview
* Role: Group Leader & Designer
* Module: CIT1062 | Graphics, Multimedia, and Social Media
* Institute: IMBS Green Campus
* Tools Used: Adobe Photoshop, Adobe Illustrator

---

## 📄 Magazine Deliverables
The project assets and full layout documentation are available within this repository:

* 📑 Full Digital Magazine: You can view or download the complete PDF directly via [Street Bites Magazine.pdf](./Street%20Bites%20Magazine.pdf).
* 📁 Source Package (PSD & Pages): You can download the complete layered Photoshop files and high-resolution assets via this [Google Drive Link](https://drive.google.com/drive/folders/19nrSQFP9UVjJKxKssja9W4zbDA0Pa2tu?usp=sharing).
